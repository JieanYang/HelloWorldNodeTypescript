### typescript:

### nodemon, ts-node

### PM2

- npm install pm2 -g
- pm2 start ecosystem.config.json --watch
- pm2 start ecosystem.config.json --only HelloWorldNodeTypescript --watch
- pm2 stop ecosystem.config.json
